

# Experiment 9 – NumPy Tools for EDA

## Aim

To study and use basic **NumPy functions in Python** for array creation, manipulation, and statistical analysis.

---

## Theory

NumPy (Numerical Python) is a Python library used for **numerical computation and data analysis**. It provides **multi-dimensional arrays and mathematical functions** that allow fast operations on large datasets. NumPy is widely used in **Exploratory Data Analysis (EDA)** to perform calculations, create arrays, and analyze numerical data efficiently.

---

## Use of Functions

np.array() – Creates a NumPy array from a list.

ndim – Returns number of dimensions of an array.

shape – Shows the size of each dimension of the array.

`dtype` – Displays the data type of array elements.

`np.zeros()` – Creates an array filled with zeros.

`np.ones()` – Creates an array filled with ones.

np.eye()` – Creates an identity matrix.

`np.arange()` – Generates numbers within a given range.

`np.linspace()` – Generates evenly spaced numbers between two values.

Array addition (`a+5`) – Adds a value to each element.

Array multiplication (`b*2`)– Multiplies each element by a value.

`np.mean()` – Calculates the average of array elements.

`np.median()` – Finds the middle value of the dataset.
